# Career Advisor (Python OOP)

Sistema simples em Python orientado a objetos para organizar e analisar perfis profissionais do futuro.
Gera recomendações de carreiras e indica lacunas de competência.

## Estrutura
- `app.py` — CLI principal.
- `base.py` — modelos (Profile, Career), persistência, recomendações.
- `profiles.json` — arquivo de dados (gerado em runtime).
- `export_profiles.json` — exportação manual via CLI.

## Requisitos
- Python 3.8+
- Nenhuma dependência externa.

## Execução
```bash
python app.py
